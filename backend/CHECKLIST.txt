APRÈS DÉPLOIEMENT - VÉRIFICATIONS SUR LE SERVEUR
═══════════════════════════════════════════════════════════════════════

☐ Backend tourne?
  ☐ ps aux | grep '[u]vicorn'
  ☐ Résultat attendu: Une ligne avec "uvicorn server:app"

☐ Port 8001 écoute?
  ☐ netstat -tlnp | grep 8001 (ou: ss -tlnp | grep 8001)
  ☐ Résultat attendu: Une ligne avec ":8001"

☐ Backend répond?
  ☐ curl http://localhost:8001/api/health
  ☐ Résultat attendu: JSON response ({"status":"healthy"} ou similaire)

☐ Variables d'environnement correctes?
  ☐ grep MONGO_URL /app/backend/.env
  ☐ Résultat attendu: mongodb+srv://...
  ☐ grep JWT_SECRET_KEY /app/backend/.env
  ☐ Résultat attendu: lottolab-secure-production-key... (PAS VOTRE-CLE)

☐ Logs sans erreur?
  ☐ tail -30 /var/log/lottolab-backend.log
  ☐ Rechercher: ERROR, Exception, Failed, error

TROUBLESHOOTING - SI PROBLÈME
═══════════════════════════════════════════════════════════════════════

❌ Je vois toujours HTTP 500
───────────────────────────────
☐ Vérifier les logs:
  ☐ tail -100 /var/log/lottolab-backend.log | grep -i error
  ☐ Copier le message d'erreur exact

☐ Redémarrer le backend:
  ☐ pkill -9 -f uvicorn
  ☐ sleep 2
  ☐ cd /app/backend
  ☐ nohup python3 -m uvicorn server:app --host 0.0.0.0 --port 8001 --workers 2 > /var/log/lottolab-backend.log 2>&1 &
  ☐ sleep 3
  ☐ curl http://localhost:8001/api/health

❌ Port 8001 est déjà utilisé
────────────────────────────────
☐ Vérifier qui l'utilise:
  ☐ lsof -i :8001 | grep LISTEN
  ☐ ou: netstat -tlnp | grep 8001

☐ Tuer le processus:
  ☐ PID=$(lsof -i :8001 | grep LISTEN | awk '{print $2}')
  ☐ kill -9 $PID

☐ Redémarrer:
  ☐ nohup python3 -m uvicorn server:app --host 0.0.0.0 --port 8001 --workers 2 > /var/log/lottolab-backend.log 2>&1 &

❌ ModuleNotFoundError
────────────────────────
☐ Réinstaller les packages:
  ☐ pip install -r /app/backend/requirements.txt

☐ Vérifier l'installation:
  ☐ python3 -c "import fastapi, motor, jose; print('OK')"

❌ MONGO_URL not set
──────────────────────
☐ Vérifier que .env existe:
  ☐ cat /app/backend/.env
  ☐ grep MONGO_URL /app/backend/.env

☐ Si absent, copier le fichier à nouveau:
  ☐ scp /app/backend/.env seller-commission-ui@IP:/app/backend/
